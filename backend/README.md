## Setup
1. Install Dependencies
```
npm install
```
```
pip install -r requirements.txt
```

2. Configure AWS
```
aws configure
```

3. Environment Variables


4. Activate Virtual Environment (optional)
```
venv\Scripts\activate
```

5. Run Application locally
```
cd app
```
```
uvicorn main:app --reload
```

6. Deploy service
```
serverless deploy
```
